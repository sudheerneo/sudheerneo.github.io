BMI Check (Android App)

BMI Check is a simple, beautiful and offline Body Mass Index (BMI) calculator built with React Native. Calculate your BMI instantly using your height, weight, age and gender. The app is lightweight, fast and designed with a modern Material Design interface for a smooth user experience.

Features

  • ✔ Instant BMI Calculation
  • ✔ Works Completely Offline
  • ✔ Beautiful Material Design UI
  • ✔ Smooth Animations
  • ✔ Supports Male & Female
  • ✔ No Ads
  • ✔ No Login Required
  • ✔ No Internet Required
